#栈(Stack)
##后进先出

```java
/**
 * 模拟堆栈数据结构 后进先出
 */
public class Stack {

    private LinkedList linkedList = new LinkedList();

    /**
     * 压栈
     *
     * @param o
     */
    public void push(Object o) {
        linkedList.add(o);
    }

    /**
     * 查看栈顶部的对象,但不移除它
     *
     * @return Object
     */
    public Object peek() {
        return linkedList.getLast();
    }


    /**
     * 移除堆栈顶部的对象，并作为此函数的值返回该对象。
     *
     * @return Object
     */
    public Object pop() {
        return linkedList.removeLast();
    }

    /**
     * 查看栈结构中是否还有对象的引用
     *
     * @return true or false
     */
    public boolean isEmpty() {
        return linkedList.isEmpty();
    }

    /**
     * 栈结构中元素的个数
     *
     * @return int
     */
    public int size() {
        return linkedList.size();
    }

}
```

#队列(Queue)
##先进先出

```Java


/**
 * 模拟队列数据结构 先进先出
 */
public class Queue {

    private LinkedList linkedList = new LinkedList();

    /**
     * 向队列中添加一个元素
     *
     * @param o
     */
    public void put(Object o) {
        linkedList.add(o);
    }

    /**
     * 返回队列中最底部的对象引用，并且将其重队列中移除
     *
     * @return Object
     */
    public Object get() {
        return linkedList.removeFirst();
    }

    public int size() {
        return linkedList.size();
    }

    public boolean isEmpty() {
        return linkedList.isEmpty();
    }

}

```
